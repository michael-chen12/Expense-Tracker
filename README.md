# Expense Tracker (Ledgerline)

A frontend-only Next.js expense tracker that stores data in `localStorage`.

## Project structure
- `client/` Next.js app
- `server/` legacy Express API (not used in the frontend-only version)

## Local development
```bash
cd /Users/chenqinfeng/Desktop/Projects/expense-tracker/client
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes
- Data is stored locally in the browser under the key `ledgerline.expenses`.
- Clearing site data will reset the list.
